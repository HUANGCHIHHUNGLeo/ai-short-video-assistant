### 商業模式說明

這不是一般的「課程」，而是「終身合夥關係」：

| 項目 | 內容 |
|------|------|
| **1 對 1 陪跑** | 手把手教到會 |
| **終身成長分享** | 師父未來學到的都會分享給徒弟 |
| **合作分潤** | 一起合作的專案，師父抽 5-10% |
| **本質** | 不是買課程，是買「成為徒弟」的資格 |

### 一般課程 vs 終身徒弟制

| 一般課程 | 終身徒弟制 |
|----------|------------|
| 上完就結束 | 終身陪跑 |
| 學完自己想辦法 | 持續分享成長 |
| 沒有後續關係 | 合作分潤機制 |
| 買的是「知識」 | 買的是「進入圈子的資格」 |

### 分潤模式的雙贏設計

**對師父的好處：**
- 被動收入（徒弟做的專案抽 5-10%）
- 擴大產能（一個人做不完，徒弟幫忙做）
- 利益綁定（徒弟成功 = 師父也賺）

**對徒弟的好處：**
- 降低風險（有師父帶，少走彎路）
- 持續學習（師父成長，徒弟也跟著成長）
- 合作機會（可能接到師父介紹的案子）
- 90-95% 還是自己的（分出去的換到更高價值）
